## 제품 이미지 (products 폴더)

제품 이미지는 각 테마/카테고리별로 다음 폴더에 저장합니다:

```
/static/images/products/{테마}/{카테고리}/
```

### 제품 이미지 네이밍 규칙
- **썸네일**: `{제품코드}.jpg` (400x400)
- **메인 이미지**: `{제품코드}-main.jpg` (800x800)
- **착용샷**: `{제품코드}-wear.jpg` (800x800)
- **상세 이미지**: `{제품코드}-detail-{번호}.jpg` (800x800)

### 예시:
```
/static/images/products/traditional/earring/
├── TRAD-EAR-001.jpg           # 썸네일
├── TRAD-EAR-001-main.jpg      # 메인 이미지
├── TRAD-EAR-001-wear.jpg      # 착용샷
├── TRAD-EAR-001-detail-1.jpg  # 상세 1
└── TRAD-EAR-001-detail-2.jpg  # 상세 2
```
